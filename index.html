Pratique - Création d'un fichier HTML

Vous êtes un grand passionné de musique, et vous souhaitez afficher des informations sur vos artistes préférés sur un site web.

Votre site devra contenir à minima :

  • Une sémantique respectée
  • Un menu de navigation avec 3 artistes
  • Une section par artiste comprenant
    1. Le nom de l'artiste
    2. Une petite description
    3. Une photo du groupe
    4. Une liste d'information contenant vos 5 morceaux préférés
    5. Un lien hypertexte emmenant vers votre clip préféré (YouTube)
